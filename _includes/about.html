Guess the Number Game is a game app for players with following features:

  1. Let the player choose a random number between 1 and 100.
  2. Challenge the player to guess the number in maximum 10 times.
  3. In each turn the player is told if the guess is right or wrong.
  4. If the guess is wrong, it shows whether the guess was too low or too high.
  5. It also displays the number previously guessed.
  6. The game will end once the player guesses correctly.
  7. The game will also end when the player runs out of turns.
  8. However, when the game ends, the player is given an option to start the game again.
